# GradeCalcApp Java Web Application

A Java Servlet based web application that calculates student grades based on marks entered through an HTML form.

This project demonstrates form handling, servlet processing, and dynamic result generation using Java and Apache Tomcat.

---

## Features

- Student grade calculation
- HTML form input handling
- Dynamic result display
- Servlet based backend processing
- Simple web application deployment

---

## Technologies Used

- Java
- Servlet API
- HTML5
- Apache Tomcat

---

## Project Structure

```text
GRADECALCAPP/
│
├── calc.html
│
└── WEB-INF/
    ├── web.xml
    └── classes/
        └── GradeServlet.java
```

---

## How It Works

1. User enters marks in the HTML form
2. Form data is sent to the servlet
3. Servlet processes marks
4. Grade/result is calculated
5. Output is displayed in browser

---

## How to Run

1. Install Apache Tomcat
2. Copy project into Tomcat `webapps` folder
3. Start Tomcat server
4. Open browser and run:

```text
http://localhost:8080/GRADECALCAPP
```
